# COPOMO-学習継続支援型コミュニティ-

## サイト概要

### サイトテーマ
「集中」「意欲」「継続」の3つの要素からポモドーロテクニックを活用した学習継続支援型のコミュニティサイト

### テーマを選んだ理由
自発的な学習の継続は、明確な目標や強い意志がなければ難しいと考えています。
特に多忙な社会人にとって限られたプライベートの時間の中で学習を継続できないことが大きな課題です。
個人で行う学習は孤独であり、つまらなく感じやすく、楽な方に逃げてしまいやすくなることが原因と考えました。
そこで、他者と共通の目標を持つことで意欲が高まり、楽しさややりたいと思う気持ちが生まれる環境が必要と考えました。
個人学習ではなく、集団で学習時間を共有することで相乗効果を生み、継続しやすい環境を提供することを目的としています。当課題を解決するために、このようなポートフォリオを制作してみようと考えました。

### ターゲットユーザ
- ​学習意欲や目標があり、楽しく学習をしたい人
- 学習意欲や目標はあるが、途中で心が折れてしまいやすい人
- 時間対効果を最大化する学習をしたい人

### 主な利用シーン
- 資格勉強や試験勉強を始めるとき
- 個人学習で心が折れそうになったとき
- 集中して学習に励みたいとき
